hexo的基本设置与操作流程

基本设置

  1. 更改主题
    #_config.yml
theme:xxx
  2. 更改默认布局为草稿(draft)
    #_config.yml
default_layout: draft

操作流程

  1. 新建草稿
    hexo new "文件名"
  2. 编辑
    #source/draft/文件名.md
  3. 移动草稿到要发布的文章类别中
    hexo publish "file"
  4. 文章生成
    hexo g
  5. blog本地预览
    hexo s
  6. blog 更新到网络
    hexo d
